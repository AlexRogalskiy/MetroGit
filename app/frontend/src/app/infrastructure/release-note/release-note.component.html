Release Notes 0.2.0

Highlights

  • File Details View - Now MetroGit has built-in file viewer so you can easily see changes for each commit. Need to see more than one file? No problem, just click and a separate window will open, allowing you to see changes side by side
  • Keyboard Shortcuts - Speed up your work! Literally, MetroGit now has keyboard shortcut for commonly used operations such as push, pull. Press ? to see all available keyboard shortcuts
  • Tags Management - Create and delete tags is now supported in MetroGit. Right click on a commit in the map to create a tag and right click a tag in the map or in the list to delete it
  • Stash Management - Now you can pop, apply or delete a specific dash via right-click menu in the map
  • Reset - MetroGit supports hard and soft reset, just right click on a commit in the map and select the reset type
